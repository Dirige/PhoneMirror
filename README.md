# PhoneMirror - 手机镜像

车机端通过局域网远程控制手机的投屏方案。

## 功能

- 手机端屏幕采集 + H.264 硬编码推流
- 车机端悬浮窗解码显示
- 支持自由缩放窗口（等比缩放）
- 触控事件回传（基于无障碍服务）
- 局域网 UDP 自动发现设备
- 深色主题 UI

## 使用方法

### 1. 手机端（发射端）
1. 安装 
2. 打开 app，点击「开始投屏」
3. 授权屏幕采集权限

### 2. 车机端（接收端）
1. 安装 
2. 打开 app，输入手机 IP 地址或点击「扫描局域网设备」
3. 点击「连接投屏」
4. 授权悬浮窗权限

### 3. 开启触控回传（可选）
在手机端设置中开启「手机镜像-发射端」的无障碍服务。
